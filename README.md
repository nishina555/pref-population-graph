## 技術スタックについて
個人ブログの[ゆめみのフロントコーディング試験のデモアプリをReactで実装しました](https://nishinatoshiharu.com/create-yumemi-front-demoapp/)で紹介しています。

## 起動方法

```bash
npm run dev

yarn dev
```
